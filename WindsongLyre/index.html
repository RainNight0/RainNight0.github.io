关于

这个项目我大概做了半个月。起初是跟群友聊天的时候聊到有些群友因为入坑晚没能拿到琴，然后有个群友提议可以做成web版然后大家都能玩了，后来我就试着做了一下，结果就一直做到了最后。(不得不说敲键盘是真的上头)

我会的不多，期间在网上找了不少文章和教程，后来在 gayhub github 发现了 染樱 大佬做的琴，因为本来做出来只是群内部玩的，所以就直接借鉴(copy)了大佬的JavaScript代码。然后经历了各种bug终于做出来了(还是没能解决按钮没法退出全屏的问题，所以大家自己手动退吧~欸嘿~)
